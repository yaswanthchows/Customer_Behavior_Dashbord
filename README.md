# Customer_Behavior_Dashbord
Built a Customer Shopping Behavior Analysis project using Python, MYSQL Workbench, and Power BI. Performed data cleaning, SQL analysis, and created interactive dashboards with KPIs, charts, and filters to visualize customer trends, product performance, and revenue insights for data-driven business decisions.
## Overview
This project analyzes customer shopping behavior using transactional data to uncover insights into purchasing patterns, customer preferences, and business performance. The workflow includes data cleaning, exploratory data analysis (EDA), SQL-based business analysis, interactive dashboard creation, and reporting to support data-driven decision-making.

## Dataset

- **Source:** Customer Shopping Behavior Dataset
- **Records:** 3,900 customer transactions
- **Features:** Customer demographics, product categories, purchase amount, subscription status, discounts, shipping type, ratings, and purchase history.

## Tools & Technologies
- **Python:** Pandas, NumPy, Matplotlib, Seaborn
- **MySQL:** Data storage and business query analysis
- **Power BI:** Interactive dashboard and data visualization
- **Gamma:** Presentation (PPT) creation
- **MS Word:** Project report documentation


## Project Steps
1. Loaded and explored the dataset using Python.
2. Performed data cleaning and preprocessing.
3. Conducted Exploratory Data Analysis (EDA).
4. Imported the cleaned dataset into MySQL.
5. Executed SQL queries to answer business questions.
6. Built an interactive Power BI dashboard.
7. Prepared a project report and presentation using Gamma.

## Dashboard
The Power BI dashboard includes:
- Revenue and Sales KPIs
- Customer Segmentation
- Product Category Analysis
- Subscription Insights
- Purchase Trends
- Interactive Filters and Slicers

## Results
- Identified customer purchasing trends and spending behavior.
- Analyzed top-performing products and categories.
- Evaluated subscription and discount impact on sales.
- Generated actionable business insights through SQL and Power BI.

## How to Run
1. Clone this repository.
2. Install the required Python libraries.
3. Run the Python notebook/script for data preprocessing and EDA.
4. Import the cleaned dataset into MySQL.
5. Execute the SQL scripts.
6. Open the Power BI (.pbix) file to explore the dashboard.
7. Refer to the report and presentation for detailed findings.
